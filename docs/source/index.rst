Introduction
~~~~~~~~~~~~~
A/B testing is process which allows developer/data scientist to analyze and evaluate, the performance of products in an experiment. 
In this process two or more versions of a variable (web page, page element, products etc.) are shown to different segments of website 
visitors at the same time to determine which version leaves the maximum impact and drives business metrics.

In A/B testing, **A** refers to the original testing variable. 
Whereas **B** refers to a new version of the original testing variable. Impact of the results can be evaluated based on,

* Conversion Rate
* Significance test
